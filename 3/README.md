# 3пр Пешеходный переход

На основе кода решения 2й практики (в папке src/), сделайте имитацию светофоров на однополосной дороге с регулируемым пешеходным светофором. Условия:
1) Отсчёт должен быть внутри классов (не в loop() и не в setup()).
2) Синхронизируемые светофоры (пешеходный от автомобильного) должен иметь указатель на свойство класса светофора, от которого он зависит и проверять его каждую секунду (отсчёт внутри класса).
3) В setup() и в loop() только конфигурация и создание экземпляров классов.
