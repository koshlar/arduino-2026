# 2пр Разделение логики

1) Сделайте простые светофоры (без дисплея, просто 2-3 светодиода в зависимости от вида).
2) Разделите логику (загорание отдельно, отдельная функция для проверки).
4) Заверните это всё в класс TrafficLight и сделайте 2 дочерних класса для светофоров: PedestrianTrafficLight для пешеходного и CarTrafficLight для автомобильного.
5) Проверка нынешнего цвета должна быть реализована через свойство объекта класса. Например:

```cpp
CarTrafficLight carTrafficLight = new CarTrafficLight(...)

if(carTrafficLight.currentLight == "red") {
  ...
} ...
```

6) Добавьте светодиод, который меняет цвет синхронно смене цвета у светофоров.
